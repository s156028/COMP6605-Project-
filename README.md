# COMP6605-Project-
Voltage-Stratified Data Analytics Study of Electrical Distribution Outages

## Data Availability

The original NEDC outage dataset used in this study is proprietary and cannot be publicly shared due to the terms of the No-Objection Letter (NOL) granted by Nama Electricity Distribution Company (NEDC).

Researchers seeking independent access to the original dataset must submit a formal request directly to NEDC.

The public repository therefore includes:
- analysis notebooks,
- reproducibility scripts,

For academic access inquiries, researchers should contact:
Nama Electricity Distribution Company (NEDC), Oman.
